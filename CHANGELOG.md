# Histórico de alterações

Todas as alterações notáveis deste projecto são documentadas neste ficheiro.

O formato inspira-se em [Keep a Changelog](https://keepachangelog.com/pt-PT/1.0.0/), e a versão segue a do ficheiro `package.json`.

## [1.1.0] — 2026-04-25

### Documentação e publicação

- README alinhado ao GitHub: stack técnica, requisitos de sistema, guia de build.
- Pasta `installer/` com instruções para copiar o instalador Windows a partir de `dist/`.
- Este ficheiro de histórico de versões.

### Funcionalidades (aplicação)

- Notas em **modo canvas** (arrastar e organizar) e **modo abas**.
- **Editor de texto rico** (Tiptap) com formatação, listas, código, tarefas, etc.
- **Sistema de plugins** extensível.
- **Exportação** (HTML, Markdown, JSON, entre outros) e **auto-guardar**.
- Atalhos de teclado e integração **desktop** via Electron (diálogos, ficheiros, pasta de backup em documentos).
